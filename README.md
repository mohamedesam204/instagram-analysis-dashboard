# instagram-analysis-dashboard
لوحة معلومات تفاعلية لتحليل أداء حسابات إنستجرام، بتعرض مؤشرات أساسية مثل عدد المتابعين، الإعجابات، معدل التفاعل، الوصول والانطباعات. الدashboard بتوضح أعلى فئات المحتوى من حيث الوصول، مصادر الزيارات، نمو المتابعين شهريًا، وتأثير عدد الهاشتاجات وطول الكابشن على الوصول، مع إمكانية التصفية حسب نوع المحتوى (Reel, Video, Photo, Carousel).
